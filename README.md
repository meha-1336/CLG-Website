🎓 College Application Form
📖 Project Description

The College Application Form is a simple web-based form where students can enter their details to apply for admission.
It collects important information like student name, email, phone number, course selection, and address.

The form is designed to be user-friendly, responsive, and easy to use on all devices such as mobile, tablet, and desktop.
